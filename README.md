# 代码生成工具
后端项目。采用NodeJS开发，MySQL数据库，以及众多中间件。实现一个可连接指定数据库，并按照指定模板生成指定数据表的前端代码。

| 功能模块  | 功能点  |
| ------------ | ------------ |
|  数据库信息 | 数据库连接信息、驱动类等  |
| 生成配置  | 包名称、表前缀、模块信息等  |
| 模板配置 | 配置模板相关的信息 |
| 代码生成  | 展示数据库信息以及表信息提供代码生成功能 |

### 本文件夹包含的内容

------------

**1.基于NodeJS的后端服务**

	service文件夹

**2.基于Vue的前端展现**

	web文件夹
